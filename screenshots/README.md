## Screenshot Guidelines

- **Resolution:** At least 1920x1080 for desktop, 1080x1920 for mobile
- **Privacy:** Blur or redact player names if needed
- **Branding:** Use consistent Roll20 dark theme
- **Clarity:** Show clear examples of the feature in action
- **Real Play:** Use actual game scenarios, not lorem ipsum

## Format
- PNG for UI screenshots (better quality)
- JPG for photos of physical setups
- GIF for animated workflows (optional but powerful)

## How to Capture

### Roll20 Chat
1. Have a clean chat window showing AI narration
2. Use browser screenshot tools (Firefox/Chrome DevTools)
3. Crop to focus on the narration box

### Relay Page
1. Show a processed queue with responses
2. Highlight the copy buttons
3. Include status indicators

### Turn Queue
1. Show multiple players in queue
2. Demonstrate the "next" command flow
